This is tcss example page.

tcss is a css file that provides useful simple css classes. I wrote this because i was writing same stuff again and again everytime.
I often use this with bootstrap but it works just as well without it.

tcss GitHub

Wrapper

tcss add .wrapper class that adds a basic wrapper for centered content. All of this content you are reading are centered on the page with it. On mobile or smaller screensizes it takes the fullpage width. Wrappers width can be easily modified with --wrapper-width variable.

Simple Box

Create a simple box by giving an element .box class.

Box with rounded edges.

Add a small rouneded corners to a box with .box-rounded

Box hover effect.

With .hoverable class you can add simple hover effect to an element.

Pattern and color

You can apply color and background pattern to boxes. (Pattern images are still in production).

This is a section

When using section tags small border is added top and bottom of them to seperate content more clearly. This can be removed by changing --item-section-border variable to 0px





To add proper spacing to an image apply .timage class. This gives same margins as in .box and inputs.
To add an shadow filter to a image by apply .timage-float

Add simple image hover effect with .timage-hover

Ledge arrow hover effect.

Ledge arrow effects can be added by applying class clp-tri-right Other valid directions are right,left,up and down.

Third image also has .timage-hover class for extra hover effect.

Hidable content

There are also classes for hiding things on smaller screensizes or desktop. .hide-mobile and .show-mobile. Make your browser smaller to see the effect.
